# Coding (rename me!)

사람들의 코딩 지식 향상을 폭발적으로 도와주는 사이트

## Requirements

- python 3.5+ (prefer latest)

## Project Components

- `core`: Shared python model for backend
- `api`: Main API server
- `treadmill`: Sandboxed untrusted code executor
- `ws`: Realtime websocket server
